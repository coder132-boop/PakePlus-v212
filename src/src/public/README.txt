ChoreCore - Public Assets Folder

Place your app icons here:

1. icon.png
   - Size: 512x512 pixels or larger (1024x1024 recommended)
   - Format: PNG with transparency
   - Used for: App icon on all platforms

2. tray-icon.png
   - Size: 32x32 pixels
   - Format: PNG with transparency  
   - Used for: System tray icon

3. favicon.ico (optional)
   - Size: Multiple sizes (16x16, 32x32, 48x48)
   - Format: ICO
   - Used for: Browser tab icon

How to create icons:
- Use the ChoreCore logo (teal/cyan gradient)
- Online tools: iconverticons.com, cloudconvert.com
- Or use ImageMagick: convert logo.png -resize 512x512 icon.png

Placeholder icons will work for testing, but add proper branding before distribution!
